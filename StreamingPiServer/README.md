StreamingPiServer
============

Puerto por defecto (1024):
```Shell
pi@raspberrypi ~ $ java -cp ./StreamingPiServer-1.0-SNAPSHOT.jar 
```

Con puerto:

```Shell
pi@raspberrypi ~ $ java -cp ./StreamingPiServer-1.0-SNAPSHOT.jar 2048
```

Para salir, presione Ctrl + C
